The vertical-align Property

vertical-align은 오로지 inline과 table-cell 엘리먼트에만 적용된다.
block level 엘리먼트에는 사용할 수 없다.

vertical-align: baseline (default):

셀의 baseline을 같은 행의 다른 cell들의 baseline과 동일하게 정렬합니다.

A: An image IMAGE with a default alignment.

vertical-align: text-top:

B: An image IMAGE image with a text-top alignment.

vertical-align: top:

BB: An image IMAGE image with a top alignment.

vertical-align: middle

M: An image IMAGE with a default alignment.

vertical-align: text-bottom:

C: An image IMAGE image with a text-bottom alignment.

vertical-align: sub:

D: An image IMAGE image with a sub alignment.

vertical-align: super:

E: An image IMAGEimage with a super alignment.
