JavaScript FUNCTIONS 

지역함수(local function) : 함수 안에 정의된 내장함수를 의미
-포함된 함수 안의 블록사이에서만 쓰일 수 있음
-포함된 함수의 실행이 종료되면 지역함수도 소멸됨
-함수 내부에서 선언된 변수명과 변수값은 함수 외부에서 사용할 수 없음

전역함수(global function): 여러 코드 블록에서 동시에 쓰일 수 있는 함수를 말한다
-전역변수: 여러 코드 블록에서 동시에 쓰일 수 있는 변수를 말한다
함수 밖에서 선언된 변수의 경우 함수 내부의 코드 블록에서도 동시에
사용할 수 있다
